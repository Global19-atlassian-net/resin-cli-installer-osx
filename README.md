Resin CLI Installer OS X
------------------------

The [Resin CLI](https://github.com/resin-io/resin-cli) OS X installer.

Prequisites for building
------------------------

- OS X.
- `curl`

Building
--------

Running `make` will create an installer into `build/resin-cli-setup.pkg`:

```sh
$ make
```
